#### 使用说明

1.  打开后端项目下载好所需的maven依赖，然后看application.yml，更改自己的端口（虚拟机ip）
2.  安装好自己的mysql，把账号密码设置为root，admin
3.  运行后端项目
4.  打开前端项目，使用npm i初始化依赖，然后安装所需的依赖
5.  在main.js中修改自己的ip以及端口号
6.  运行前端项目npm run serve
